# Hunkybot
My personal discord bot

## Installation
```
git clone https://github.com/haroldpark/hunkybot
cd hunkybot
npm install
```

## Usage
```
node server.js
```

## Built With
* Javascript
* [Discord.js](https://discordapp.com/developers/docs/intro) - Cloud-powered communication API
